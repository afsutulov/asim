# mosaic

`mosaic` — новый Go-проект для построения RGB-мозаики Sentinel-2 L1C по Пермскому краю.

## Что делает

1. Читает `config.json` (`--config`, по умолчанию `./data/config.json`).
2. Берёт из него:
   - `sentinel` — корень архива Sentinel-2 L1C;
   - `results_path` — куда сохранить результат;
   - `logs` — куда писать лог `mosaic.log`.
3. Получает из CLI:
   - `--start` — начало периода (`YYYY-MM-DD`);
   - `--end` — конец периода (`YYYY-MM-DD`);
   - `--out` — имя выходного файла без расширения.
4. Читает `cash.json` в годовых папках, попадающих в период.
5. Для каждого tile из списка Пермского края выбирает **самый поздний** снимок с `cloud == 0` внутри периода.
6. Собирает RGB из каналов `B04/B03/B02`.
7. Нормализует цвет по общим статистикам, чтобы уменьшить видимые «кубики» между тайлами.
8. Делает геопривязанную мозаику без смещений и сохраняет её в `results_path/<out>.jp2`.
9. Ведёт полный лог в `mosaic.log`.

## Зависимости

Проект написан на Go, но для сборки RGB и мозаики использует **GDAL command line tools**:

- `gdalinfo`
- `gdalbuildvrt`
- `gdal_translate`
- `gdalwarp`

Они должны быть доступны в `PATH`.

## Предполагаемая структура Sentinel-2 L1C

Проект `cron` индексирует `.SAFE` каталоги и кладёт в `cash.json` путь `img_data_path`, который обычно указывает на:

```text
<sentinel>/<year>/<SAFE>.SAFE/GRANULE/<granule>/IMG_DATA
```

Для Sentinel-2 L1C проект ищет RGB-каналы по шаблонам:

- `*_B04*.jp2`
- `*_B03*.jp2`
- `*_B02*.jp2`

## Пример запуска

```bash
go run . \
  --config ./data/config.json \
  --start 2025-06-01 \
  --end 2025-06-30 \
  --out perm_2025_06
```

Результат:

```text
<results_path>/perm_2025_06.jp2
```

Лог:

```text
<logs>/mosaic.log
```

## Замечания

- Если период пересекает несколько лет, проект читает `cash.json` по каждому году из диапазона.
- Если для какого-то tile нет снимка с `cloud == 0`, проект завершится ошибкой и перечислит отсутствующие tile.
- Для нормализации используется линейное приведение каждого тайла к общим средним/стандартным отклонениям по каналам. Это не заменяет полноценный radiometric balancing, но заметно уменьшает различия между тайлами.
